# 都道府県別 人口推移分析アプリ

## アプリの概要
政府統計の総合窓口（e-Stat）のデータを使用し、都道府県ごとの人口推移を可視化・比較できるWebアプリケーションです。
サイドバーから複数の都道府県を選択し、時系列の推移（折れ線グラフ）と最新年の比較（棒グラフ）を行うことができます。

## 使用したデータ
- **統計表名**: 社会・人口統計体系
- **データセット名**: 都道府県データ（社会生活統計指標）
- **データ取得日**: 2026年1月26日
- **各列の意味**:
  - `都道府県`: データの対象となる地域名
  - `西暦`: 調査が行われた年（西暦）
  - `人口`: 総人口（単位：人）

## 使用した未使用UI部品
授業では扱っていない以下のUI部品を使用し、アプリの視認性と操作性を向上させました。

1. **`st.caption`**
   - **目的**: タイトル直下にデータの出典を控えめに表示し、情報の信頼性を担保するため。
   - **使用箇所**: アプリタイトルの直下。

2. **`st.divider`**
   - **目的**: 条件設定エリア（サイドバーの操作結果）と、可視化エリア（グラフ）の境界を視覚的に明確にするため。
   - **使用箇所**: データのフィルタリング処理とグラフ描画処理の間。

3. **`st.expander`**
   - **目的**: 詳細なデータテーブルは常時表示すると画面を占有してしまうため、必要な時だけ開けるようにするため。
   - **使用箇所**: ページ下部の「データ詳細を確認する」セクション。

## 参考にしたアプリ (Gallery)
- **アプリタイトル**: Streamlit Gallery "Sales Dashboard"
- **参考にした点**: 画面をカラム（`st.columns`）で分割し、異なる種類のグラフ（推移とランキング）を並べて表示する構成。
- **反映した点**: 画面を左右2分割し、左側に時系列の「折れ線グラフ」、右側に最新時点の「棒グラフ」を配置して比較しやすくしました。
