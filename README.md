# 都道府県別 人口推移分析アプリ

## アプリの概要
政府統計の総合窓口（e-Stat）のデータを使用し、2005年から2024年にかけての都道府県ごとの人口推移を可視化・比較できるWebアプリケーションである。
サイドバーから複数の都道府県を選択し、時系列の推移（折れ線グラフ）と最新年の比較（棒グラフ）を行うことができる。

## 使用したデータ
- **出典**: 政府統計の総合窓口(e-Stat) 社会・人口統計体系
- **対象期間**: 2005年〜2024年
- **データ項目**:
  - `都道府県`: 地域名
  - `西暦`: 対象年度
  - `人口`: 総人口（単位：人）

## 使用した未使用UI部品
授業では扱っていない以下のUI部品を使用し、アプリの視認性と操作性を向上させた。

1. **`st.caption`**: タイトル直下にデータの出典を控えめに表示するため使用。
2. **`st.divider`**: 条件設定エリアとグラフ表示エリアの境界を明確にするため使用。
3. **`st.expander`**: 詳細なデータテーブルを折りたたみ表示にし、画面を整理するため使用。
