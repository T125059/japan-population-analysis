# 都道府県別 人口推移分析アプリ

## アプリの概要
政府統計の総合窓口（e-Stat）のデータを使用し、2005年から2024年にかけての都道府県ごとの人口推移を可視化・比較できるWebアプリケーションである。
サイドバーから複数の都道府県を選択し、時系列の推移（折れ線グラフ）と最新年の比較（棒グラフ）を行うことが可能である。

## 使用したデータ
- **出典**: 政府統計の総合窓口(e-Stat)
- **統計名**: 人口推計 各年10月1日現在人口 令和２年国勢調査基準
- **表番号**: 005「都道府県，男女別人口－総人口，日本人人口」
- **対象期間**: 2005年 〜 2024年

## 工夫した点
基本的なグラフ表示に加え、アプリを見やすくするために以下の工夫を行った。

1. **人口情報の強調 (`st.metric`)**
   グラフだけでなく、具体的な数値も一目でわかるように、選択した地域の「総人口」と「前年比」を画面上部に大きく表示した。

2. **画面レイアウトの整理 (`st.expander` / `st.divider`)**
   - 詳細なデータ表は場所を取るため、必要な時だけ開けるように折りたたみ式にした。
   - 条件設定とグラフの間に区切り線を入れて、見やすく整理した。

3. **出典の明記 (`st.caption`)**
   データの信頼性を示すため、タイトル下にデータの出典元を正確に記載した。
