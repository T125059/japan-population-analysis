# 都道府県別 人口推移分析アプリ

## アプリの概要
政府統計の総合窓口（e-Stat）のデータを使用し、2005年から2024年にかけての都道府県ごとの人口推移を可視化・比較できるWebアプリケーションである。
サイドバーから複数の都道府県を選択し、時系列の推移（折れ線グラフ）と最新年の比較（棒グラフ）を行うことが可能である。

## 使用したデータ
- **出典**: [政府統計の総合窓口(e-Stat) 社会・人口統計体系](https://www.e-stat.go.jp/dbview?sid=0003448232)
- **対象期間**: 2005年 〜 2024年
- **データ項目**:
  - `都道府県`: 地域名
  - `西暦`: 対象年度
  - `人口`: 総人口（単位：人）

## 実装した機能と工夫点
授業で扱った基本的なグラフ表示に加え、分析ツールの実用性を高めるため、以下の機能を独自に追加実装した。

1. **重要指標の可視化 (`st.metric`)**
   選択した地域の「最新の総人口」と「前年比の増減数」を数値で強調表示した。
   グラフを見る前に、人口が増加傾向にあるか減少傾向にあるかを直感的に把握できるようにするためである。

2. **データの二次利用 (`st.download_button`)**
   フィルタリングした結果をCSVファイルとしてダウンロードできる機能を実装した。
   画面上で確認するだけでなく、データを手元に保存して活用できるように利便性を考慮した。

3. **視認性の向上 (`style` / `st.divider`)**
   - データフレーム内の数値をカンマ区切り（例: 12,000,000）で表示し、桁数の多い人口データでも読み間違えを防ぐようにした。
   - `st.divider` を使用して条件設定と分析結果の境界を明確にし、情報の区分けを行った。

4. **情報の整理 (`st.expander` / `st.caption`)**
   - 詳細なデータテーブルは場所を取るため、必要な時だけ展開できる折りたたみ形式にした。
   - タイトル直下にデータの出典元リンクを明記し、データの信頼性を担保した。
